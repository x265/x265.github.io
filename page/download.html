Linux 64bit (Ubuntu 14.04, Linux Mint 17, Kali Linux 1.0.8 or Any latest distribution)

Ubuntu PPA
If your OS was Ubuntu 14.04 and above, use upbuntu PPA, simply copy this command and paste into terminal

sudo add-apt-repository -y ppa:upubuntu-com/multimedia && sudo apt-get update && sudo apt-get install ifme

Thanks to upbuntu for providing PPA package!
If this PPA have older version, don't worry, proceed to update IFME by clicking "update" button (maybe need sudo first to avoid error), it works anywhere!

Other Linux Desktop
If you have other Linux distribution such as Debian, type this command (if you have CentOS such as, try install these package)

sudo apt-get install mono-complete mediainfo libavformat-dev libswscale-dev libboost-all-dev (also see list of shared binaries here)
To run, open ifme.sh or via terminal and type sh ./ifme.sh
